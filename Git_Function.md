#GitHub使用方法

##初始化設定
```console
git init
```
##Clone檔案到本地
```console
git clone https://github.com/YoRayr/MCUTFitness-training.git
```
##查看當前工作區
```console
git status
```
##加入暫存變更
```console
git add [filename]
```
##Commit本次修改
```console
git commit -m "your message"
```
##push上傳
```console
git push
```
##切換分支上傳
```console
git push --set-upstream origin test1
```
##新增分支branch
```console
git branch name
```
##切換分支
```console
git checkout name
```
